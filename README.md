imagic
------
Library to create [autostereogram images](http://en.wikipedia.org/wiki/Autostereogram).

Requirements:

* Go (http://golang.org/)


Run example:

    go get github.com/cartland/go/imagic
    go install github.com/cartland/go/example
    $GOPATH/bin/example
